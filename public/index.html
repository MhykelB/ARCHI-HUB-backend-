Endpoints Request Body
Login : http://archi-hub-backend.vercel.app/auth/login {email : user's email, password : user's password}
Register : http://archi-hub-backend.vercel.app/auth/register {firstName, lastName,userName,email,password,}
Create and send reset token link: http://archi-hub-backend.vercel.app/resetPassword/createToken {email : user's email}
Reset Password: http://archi-hub-backend.vercel.app/resetPassword/setNewPassword {newPassword : user's newpassword}

Retrieve token from req.params and include it in headers.authorization
